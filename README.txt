
■このプログラムの概要

    機械学習によってカタカナを学習・認識させるプログラムです。


■ファイル構成

    KatakanaRecog.ipynb
        jupyter notebook 型式のファイルです。


    ETL5C（ETL5C.zipを解凍してください）
        産業技術総合研究所の「（常用）手書カタカナ文字データベースＥＴＬ５」です。
            Webサイト：http://etlcdb.db.aist.go.jp/
            データ仕様 ：http://etlcdb.db.aist.go.jp/etlcdb/etln/form_c.htm

    BackUp.zip
        プログラムのバックアップです。動作には不要です。


■動作環境

    以下の環境で動作確認しました。

        Python          3.6.3
        scikit-learn    0.19.1
        numpy           1.13.3
        matplotlib      2.1.0
